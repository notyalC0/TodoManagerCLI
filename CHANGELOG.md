# Changelog

All notable changes to this project will be documented in this file.
Format based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.1.0] - 2026-04-08

### Added
- Task details view — access full task info by pressing `Enter`.
- Interactive search results — navigate results with arrow keys and press
  `Enter` to view details or `e` to edit directly from results.

### Changed
- Project modularized into `lib/core`, `lib/logic`, `lib/models`, and `lib/ui`.
